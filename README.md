# pwn.college Learning Repository

This repository documents my hands-on cybersecurity training through
[pwn.college](https://pwn.college/), reverse engineering practice,
binary analysis, Linux internals, and low-level debugging.

The goal of this repository is not just to collect challenge solves,
but to document the *process* of learning:
- reversing binaries
- understanding assembly
- debugging programs dynamically
- analyzing file formats
- interacting with Linux system internals
- building disciplined troubleshooting and research habits

---

# Current Focus Areas

## Reverse Engineering
- Static analysis with:
  - Ghidra
  - objdump
  - strings
  - hexdump / xxd
- Dynamic analysis with:
  - gdb
  - GEF
- x86-64 assembly analysis
- ELF internals
- File format reversing
- Magic number validation
- Control flow analysis

## Linux / Systems
- Process interaction
- File descriptors
- Pipes and redirection
- Syscalls
- Memory layout
- Stack behavior
- Permissions and SUID binaries

## Tooling
- Git / GitHub workflow
- Bash scripting
- Python automation
- SSH / SCP usage
- Linux command-line proficiency
